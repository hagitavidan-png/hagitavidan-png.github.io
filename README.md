## ✨ Overview

This project is a polished GitHub Pages website for **Hagit Avidan** — artist, painting teacher, and creator of the **7 Diamonds Method**.

The site presents art lessons in a warm, elegant, mobile-friendly format designed to make painting feel accessible, inspiring, and personal.

It is built to support learners through observation, imagination, emotion, and visual exploration.

---

## 🌍 Highlights

- **Bilingual experience** — Hebrew and English
- **Responsive design** — optimized for mobile, tablet, and desktop
- **Featured lesson experience** — a highlighted recommended lesson on the homepage
- **Expandable lesson browsing** — users can open the full lesson list when they want to explore more
- **Direct social access** — connected to Hagit Avidan’s public channels
- **Free access** — no sign-up required

## 🚀 Deployment

This website is designed for **GitHub Pages** deployment.

To publish updates:

1. Upload the updated project files to the repository root.
2. Make sure `index.html` remains in the main root directory.
3. Commit and push the changes to GitHub.
4. Open the live site via:
   - `https://hagitavidan-png.github.io`
